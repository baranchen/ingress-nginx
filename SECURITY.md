# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately:

- Open a [GitHub Security Advisory](https://github.com/baranchen/ingress-nginx/security/advisories/new)
- Or email the maintainer directly

**Do not file public issues for security vulnerabilities.**

## Supported Versions

| Version | Supported |
| ------- | --------- |
| v1.30.2 | Yes       |

## Response Time

We aim to acknowledge security reports within 48 hours and provide a fix within
7 days for critical vulnerabilities.
